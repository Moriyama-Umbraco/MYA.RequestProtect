# Moriyama.RequestProtect.Umbraco.Admin

Provides a simple admin dashboard for seeing the current configuration for Moriyama.RequestProtect.

## Features

- 🔒 Shows the enabled/disabled state
- 🌐 If enabled, shows the query string component for easy copy/paste
- ⚙️ Display the current rules configuration as loaded by the website.

## Installation

```bash
dotnet add package Moriyama.RequestProtect.Umbraco.Admin
```

Add the users who should be able to see the admin dashboard to the user group `Request Protect`. After that, they will find the dashboard in the Settings section of Umbraco.
